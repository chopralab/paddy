Paddy Python Package¶

Parameter optimization package for Python¶

Paddy is a Python package developed as an extension of the Paddy Field Algorithm, a geneteic, plant based, global optimization algorithm proposed by Premaratne et al. (2009)

Paddy was developed with the intention of being used to optimize hyperparameters of machine learning algorithms and as a general metaheuristic.
